# Calculator in C

A simple C program to perform basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- C Language
- GCC Compiler

## How to Run

1. Compile the program:
   gcc calculator.c -o calculator

2. Run the program:
   ./calculator
